**실습 예제:**

**소스 코드 예제:**

```c
int a = 5;
int b = 10;
int c = a + b * 2;
if (c > 20) {
    c = c - 5;
} else {
    c = c + 5;
}
for (int i = 0; i < 10; i++) {
    c = c + i;
}
```

**중간 코드 생성기 예제:**

```c
#include <stdio.h>

void generate_code() {
    printf("t1 = 5\n");
    printf("t2 = 10\n");
    printf("t3 = t2 * 2\n");
    printf("t4 = t1 + t3\n");
    printf("c = t4\n");
    printf("if (t4 > 20) goto L1\n");
    printf("c = t4 + 5\n");
    printf("goto L2\n");
    printf("L1: c = t4 - 5\n");
    printf("L2:\n");
    printf("for (int i = 0; i < 10; i++) {\n");
    printf("    c = c + i\n");
    printf("}\n");
}

int main() {
    generate_code();
    return 0;
}
```
